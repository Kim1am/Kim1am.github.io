文章用于总结部分用到的 ES6 新 API遍历提醒：对于纯对象的遍历，for..in对于数组遍历，如果不需要知道索引，for..of 迭代更合适，因为还可以中断；如果需要知道索引，则 forEach()更合适；对于其他字符串，类数组，类型数组的迭代，for..of 更加适合

# 字符串扩展

方法 返回值 说明
String.includes(string , index) Boolean 是否找到了 string 字符串，index 为开始搜索位置
String.startsWith(string , index) Boolean string 是否在原字符串的头部,index 为开始搜索位置
String.endsWith(string , index) Boolean string 是否在原字符串的尾部,第二个参数针对前 index 个字符
String.repeat(times) String 表示将原字符串重复 times 次(times 为负数或者Infinity，会报错，NaN 等同于 0)
String.padStart(length,string) String 从字符串的头部以 string 开始补全直到字符串长度为 length ，忽略第二个参数则以空格补全
String.padEnd(length,string) String 从字符串的头部以 string 开始补全直到字符串长度为 length ，忽略第二个参数则以空格补全
let s = 'Hello world!';
//在使用第二个参数时,endsWith的行为与其他两个方法有所不同。它针对前n个字符。
//而其他两个方法针对从第n个位置直到字符串结束
s.startsWith('world', 6) // true
s.endsWith('Hello', 5) // true
s.includes('Hello', 6) // false

'x'.repeat(3) // "xxx"
'hello'.repeat(2) // "hellohello"
'na'.repeat(0) // ""
//参数如果是小数，会被取整。
'na'.repeat(2.9) // "nana"

'x'.padStart(5, 'ab') // 'ababx'
'x'.padStart(4, 'ab') // 'abax'

'x'.padEnd(5, 'ab') // 'xabab'
'x'.padEnd(4, 'ab') // 'xaba'

# 数值

方法 返回值 说明
Number.isFinite(number) Boolean 检查一个数值是否为有限的（finite），即不是Infinity。注意，如果参数类型不是数值，Number.isFinite 一律返回 false。
Number.isNaN(number) Boolean 用来检查一个值是否为 NaN, 如果参数类型不是 NaN，Number.isNaN 一律返回 false。
isFinite(25) // true
isFinite("25") // true
Number.isFinite(25) // true
Number.isFinite("25") // false

isNaN(NaN) // true
isNaN("NaN") // true
Number.isNaN(NaN) // true
Number.isNaN("NaN") // false
Number.isNaN(1) // false

它们与传统的全局方法 isFinite()和 isNaN()的区别在于，传统方法先调用 Number()将非数值的值转为数值，再进行判断;而这两个新方法只对数值有效，Number.isFinite()对于非数值一律返回false, Number.isNaN()只有对于 NaN 才返回 true，非 NaN 一律返回 false。

方法 返回值 说明
Math.trunc(number) Number 用于去除一个数的小数部分，返回整数部分。对于非数值，Math.trunc 内部使用 Number 方法将其先转为数值。对于空值和无法截取整数的值，返回 NaN。
Math.sign(number) Number 判断一个数到底是正数、负数、还是零。对于非数值，会先将其转换为数值。

Math.sign(number) 会返回五种值。 参数为正数，返回+1； 参数为负数，返回-1； 参数为 0，返回 0； 参数为-0，返回-0; 其他值，返回 NaN。

# 函数

1.函数设置参数默认值 指定了默认值以后，函数的length属性，将返回没有指定默认值的参数个数。也就是说，指定了默认值后，length属性将失真。

function log(x, y = "World") {
  console.log(x, y);
}

log("Hello"); // Hello World
log("Hello", "China"); // Hello China
log("Hello", ""); // Hello

new Function().name; // "anonymous"
function foo() {}
foo.bind({}).name; // "bound foo"

3.箭头函数（）=>

var f = v => v;

// 等同于
var f = function(v) {
  return v;
};

var f = () => 5;
// 等同于
var f = function() {
  return 5;
};

var sum = (num1, num2) => num1 + num2;
// 等同于
var sum = function(num1, num2) {
  return num1 + num2;
};

//如果箭头函数的代码块部分多于一条语句，就要使用大括号将它们括起来，并且使用return语句返回。
var sum = (num1, num2) => {
  return num1 + num2;
};

//如果返回一个对象，则需要用大括号阔起对象
let getTempItem = id => ({ id: id, name: "Temp" });

箭头函数有几个使用注意点:

  • 函数体内的 this 对象，就是定义时所在的对象，而不是使用时所在的对象。

  • 箭头函数没有 this，所以需要通过查找作用域链来确定 this 的值。

  • 不可以当作构造函数，也就是说，不可以使用new命令，否则会抛出一个错误。

  • 不可以使用 arguments 对象，该对象在函数体内不存在。如果要用，可以用 rest 参数代替。

  • 不可以使用 yield 命令，因此箭头函数不能用作 Generator 函数。(也是 ES6 新特性，后续会介绍到)

  • 除了 this，以下三个变量在箭头函数之中也是不存在的，指向外层函数的对应变量：arguments、super、new.target。
